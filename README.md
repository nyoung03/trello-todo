# trello-todo

노마드 코더 리액트 마스터 강의를 듣고 학습한 프로젝트입니다.
React Hook Form 을 이용하여 입력폼을 빌드하고 검증하는 것과 Recoil 의 atoms 와 selectors를 이용해 어플리케이션의 상태를 관리하는 법을 배웠습니다. 마지막으로 Beautiful DnD를 활용하여 드래그-앤-드랍을 구현했습니다.

### 스텍
- react
- typescript
- styled-components
- recoil
- react-hook-form
- react-beautiful-dnd
- gh-pages
